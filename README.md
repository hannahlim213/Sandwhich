# Sandwhich

Sandwich app lets you create a new sandwich, eat it and give it back.

## Getting Started

Demo video

![Homepage](images/homepage.PNG)


### Installing

* NPM init -y

* NPM install path

* NPM install mysql

*NPM install express-handlebars

* NPM install express

* NPM install body-parser
